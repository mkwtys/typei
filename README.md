> types interactive installer

## Install

```sh
npm i -g typei
```

## Usage

```sh
Usage:
  $ typei

Options:
  --update, -u   Uninteractive update. Apply all updates without prompting
  -h, --help     Show help
  -v, --version  Show version number

Examples:
  $ typei -u
```
